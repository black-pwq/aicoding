# Getting Started

本页面向刚接触 AI coding（Vibe Coding）/ Coding Agent 方向的实验室成员，目标是帮助快速建立基础概念、阅读顺序和学习路径。

## 本页建议覆盖

- 什么是 AI coding
- 什么是 Vibe Coding
- 什么是 Coding Agent
- AI coding、Vibe Coding 与 Coding Agent 之间的关系
- 建议的入门阅读顺序
- 建议优先了解的工具与 benchmark

## 推荐组织方式

### 1. 基础概念

用简洁语言说明核心术语，避免一开始陷入过多实现细节。

### 2. 入门阅读路径

先综述，再工具，再 benchmark，再研究问题。

### 3. 实践起点

给出适合实验室新成员的最小实践入口，例如先体验哪些工具、先阅读哪些项目、先了解哪些典型任务。

- 推荐先读的综述与代表性论文
- 推荐先尝试的工具和公开资源页
